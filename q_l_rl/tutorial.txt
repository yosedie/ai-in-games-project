## ðŸ“¦ 4. Create & Setup Project

### Step 1: Create a Project Folder
```bash
cd C:\Users\v\Downloads\git\ai-in-games-project
cargo new q_l_rl
cd q_l_rl
```

**Folder structure formed:**
```
q_l_rl/
â”œâ”€â”€ Cargo.toml
â””â”€â”€ src/ 
â””â”€â”€ main.rs
```

### Step 2: Edit `Cargo.toml`
Open the `Cargo.toml` file with an editor (VSCode/Notepad++), replace the contents:

```toml
[package]
name = "q_l_rl"
version = "0.1.0"
edition = "2021"

[dependencies]
bevy = "0.11"
rand = "0.8"
```

### Step 3: Copy Code to `src/main.rs`
1. Open `src/main.rs`
2. Delete all default contents
3. Paste the complete Q-Learning code (800+ lines)
4. Save the file

### Step 4: First Build (Long!)
```bash
cargo build --release
```

**This will:**
- Download Bevy dependencies (~200MB)
- Compile all libraries
- **Time: 5-15 minutes the first time**

**Progress indicator:**
```
Compiling bevy v0.11.0
Compiling wgpu v0.16.0
...
```

### Step 5: Run Games
```bash
cargo run --release
```

**Or if you want dev mode (fast compilation but slow performance):**
```bash
cargo run
```

## ðŸš¨ Common Installation Issues

### Issue 1: "error: linker `link.exe` not found"
**Fix**: Install Visual Studio Build Tools (step #2)

### Issue 2: Cargo command not found
**Fix**: Restart terminal/PowerShell after installing Rust

### Issue 3: Bevy compile error
**Fix**: Rust Updates:
```bash
rustup update
```

### Issue 4: GPU/Vulkan error while running
**Fix**: Update GPU driver to latest version
